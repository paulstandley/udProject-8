## Quick Start

```BASH
cd neighborhood-map
npm start
```

```BASH
cd neighborhood-map
yarn start
```

### For servise worker create a propduction build

You can run a production by using the following commands ``` :) ```

```BASH
npm serve -s build
```

And then

```BASH
npm run serve
```

Check localhost: 5000
